# s3-pre-signed-url

### Install Dependencies

    pip install -r requirements.txt

### Run locally

    chalice local

### Deploy

    chalice deploy

### Replace the value with on following variable

    1. bucket
    2. aws_access_key_id
    3. aws_secret_access_key
    4. region_name
    
#### Deploy

    chalice deploy

Make sure `IAM` is configured in home directory
